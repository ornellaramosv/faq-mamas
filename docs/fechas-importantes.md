# Fechas importantes

!!! info "Nota importante"
    Esta información tiene fines informativos y no reemplaza la valoración médica. Ante síntomas de alarma o dudas sobre el estado del bebé, se debe consultar con el pediatra.

## ¿Cómo llevar control de nacimientos, partos y cumpleaños?

Crear un listado por secciones: nacimientos, próximos partos, cumpleaños de mamás y cumpleaños de bebés. Actualizarlo con cada novedad confirmada.

---

## ¿Qué datos pedir para actualizar el listado?

Nombre de mamá, nombre del bebé, fecha de nacimiento o FPP, sexo si quieren usar corazones de color y cumpleaños si aplica.
