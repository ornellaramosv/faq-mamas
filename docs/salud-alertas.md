# Cuándo consultar

Consulta con el pediatra o acude a urgencias si el bebé presenta:

- Fiebre.
- Dificultad para respirar.
- Rechazo persistente del alimento.
- Vómito frecuente o en proyectil.
- Sangre en las deposiciones.
- Signos de deshidratación.
- Decaimiento marcado.
- Llanto inconsolable.
- Coloración azulada o muy pálida.
- Popó blanca tipo arcilla.
