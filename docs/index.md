# The Club Moms FAQ

Un espacio organizado para consultar preguntas frecuentes sobre lactancia, fórmula, sueño, pañales, cólicos, vacunas, desarrollo y señales de alerta.

La idea es que las mamás puedan encontrar respuestas claras sin tener que buscar entre cientos de mensajes del grupo.

<div class="quick-search-box" markdown>

**¿Qué puedes buscar?**

Prueba con palabras como: `fórmula`, `popó verde`, `cólicos`, `reflujo`, `sueño`, `tetero`, `lactancia`, `vacunas`.

</div>

</div>

<div class="alert-box" markdown>

**Importante:** esta guía tiene fines informativos y no reemplaza la valoración médica. Si el bebé presenta fiebre, dificultad para respirar, rechazo persistente del alimento, vómito frecuente, sangre en las deposiciones, signos de deshidratación, decaimiento marcado o cualquier síntoma que preocupe, se debe consultar con el pediatra o acudir a urgencias.

</div>
