# Lactancia

## ¿Es normal que la leche materna se vea separada?

Sí. La leche materna puede separarse en una parte más líquida y una capa de grasa. Esto no significa necesariamente que esté dañada. Se puede mezclar suavemente antes de ofrecerla.

## ¿Debo extraerme si estoy dando fórmula?

Si el objetivo es mantener la producción de leche materna, puede ser útil extraerse en los horarios en los que el bebé recibe fórmula. Si el objetivo es disminuir producción, este proceso debe hacerse de forma gradual para evitar congestión o dolor.
