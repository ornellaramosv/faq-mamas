# Sueño del bebé

## ¿Qué puedo hacer si mi bebé no quiere dormir siesta?

Puede ayudar revisar señales de sueño, ventanas de vigilia, ambiente tranquilo, poca luz y una rutina corta antes de dormir.

## ¿Es malo usar ruido blanco?

No necesariamente. Puede ayudar a algunos bebés, pero debe usarse a volumen bajo y ubicado lejos de la cuna.
