# Vacunas y controles

## ¿Es normal que el bebé esté molesto después de una vacuna?

Puede ser normal que esté más irritable, somnoliento o con algo de molestia local. Se debe seguir la orientación del pediatra sobre manejo de fiebre o dolor.

## ¿Cuándo consultar después de una vacuna?

Se debe consultar si hay fiebre alta persistente, dificultad para respirar, reacción alérgica, llanto inconsolable prolongado o decaimiento marcado.
